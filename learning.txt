Arrays:
    size to be fixed. need to declare with size
    syntax declaration: 
        var array_name [size]array_type
    adding elements using indexes
    eg : arr[0]= "abcd"

Slice :
    to handle array limitations. does not require size decalartion.
    Syntax: var slice_name []type
            or booking := []string{}   slice of name booking of string type with empty/no values
    adding elements using append function. it will add element at the last of the slice.
    eg : 
        booking = append(booking, "abcd")
